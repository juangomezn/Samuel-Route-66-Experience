# 🚗 Samuel — Route 66 Experience

Una experiencia web cinematográfica interactiva inspirada en el universo de **Cars**, la estética de la **Ruta 66** y la emoción de una celebración muy especial: el primer año de Ian Samuel.

No es una invitación tradicional.

Es una **experiencia digital inmersiva tipo mini película interactiva**.

---

## 🎬 Concepto

Este proyecto transforma una invitación de cumpleaños en una narrativa visual animada donde el usuario:

- Enciende motores 🚗
- Entra en una experiencia cinematográfica
- Recorre la Ruta 66 en un entorno interactivo
- Descubre la historia de Ian Samuel
- Vive una experiencia visual tipo Pixar

---

## 🛠️ Tecnologías utilizadas

- ⚛️ React 19
- ⚡ Vite
- 🎨 Tailwind CSS
- 🎬 Framer Motion
- 🎧 Audio HTML5
- 🖼️ Assets personalizados (imágenes y sonido)

---

## 🚀 Características

- 🎥 Intro cinematográfica con animación y sonido
- 🛣️ Fondo parallax inspirado en la Ruta 66
- 🚗 Animaciones tipo película (McQueen como elemento central)
- 🎟️ Ticket estilo “Copa Pistón”
- ⏳ Countdown en tiempo real
- 👶 Sección emocional dedicada a Ian Samuel
- 🌄 Experiencia visual con estética Pixar / racing
- 🔊 Sistema de audio inmersivo (motor + música de fondo)

---

## 🧠 Filosofía del proyecto

Este proyecto busca demostrar cómo una **experiencia web puede ir más allá de una interfaz tradicional**, combinando:

- Narrativa visual
- Animación cinematográfica
- Interacción del usuario
- Diseño emocional

---

## 📸 Estilo visual

- 🔴 Rojo McQueen (#ef4444)

- ⚫ Fondos oscuros tipo cine

- 🎞️ Sensación de “mini videojuego web”

## 🎯 Objetivo

Convertir una invitación de cumpleaños en una **experiencia digital memorable**, donde el usuario no solo ve información, sino que la vive.
